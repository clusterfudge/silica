About Silica Cron

Silica Cron schedules agent prompts to run automatically using cron-style expressions. It integrates with heare-developer to execute prompts and stores all data in a SQLite database.

Features
  • Create and manage reusable prompts
  • Schedule prompts with flexible cron expressions
  • Track execution history and results
  • Simple web interface for easy management
Cron Expression Examples
Expression Description
0 9 * * *Daily at 9:00 AM
*/15 * * * *Every 15 minutes
0 0 * * 1Weekly on Monday at midnight
0 8 * * 1-5Weekdays at 8:00 AM
30 14 1 * *1st day of month at 2:30 PM
Timezone Information
  • Scheduling: All cron expressions are evaluated in UTC
  • Display: Times in the interface are shown in your local timezone
  • Preview: When creating jobs, you'll see the next 5 scheduled runs in your timezone
  • Consistency: Jobs run at the same absolute time regardless of server location
Agent Models
  • Haiku: Fast and cost-effective for simple tasks
  • Sonnet: Balanced performance for most use cases
  • Sonnet 3.5: Advanced reasoning and analysis
  • Opus: Most powerful for complex tasks
Agent Types
  • Basic: General-purpose agent for various tasks
  • Research: Deep analysis and investigation focused
  • Engineer: Technical development and problem-solving
