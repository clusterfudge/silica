About Silica Cron

Silica Cron is a web application for scheduling agent prompts to run automatically using cron-style expressions. It integrates with heare-developer to execute prompts and stores all data in a SQLite database.

  • Create and manage reusable prompts
  • Schedule prompts with flexible cron expressions
  • Track execution history and results
  • Simple web interface for easy management
